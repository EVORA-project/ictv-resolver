This page is a showcase of the public ICTV Ontology API developed by the EVORA project in collaboration with ICTV, and made available through the Ontology Lookup Service (OLS). It demonstrates how programs can resolve ICTV taxonomic terms to their latest version through the public ICTV OLS API. If you are a human user, you may prefer visiting the official ICTV taxonomy interface at ictv.global/taxonomy.
Data sources © ICTV (CC BY 4.0) — retrieved using the ICTV ontology via the public OLS API with the ICTV API JS Helper from EVORA-project/ictv-ontology.
This page is a static client-side application, hosted on GitHub Pages (https://evora-project.github.io/ictv-resolver/) . No data is stored or processed on a local backend/server —
all the magic happens on the ICTV OLS API.
ICTV Taxon Resolver
Paste a virus name, a taxon name (incl. synonyms), an ICTV ID/IRI, or an NCBI Taxon ID. Choose an action, then resolve.
Tip: for NCBI, both 64320 and ncbitaxon:64320 work.
